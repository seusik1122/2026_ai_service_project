# 국내 강의 통합 추천 시스템

AI 기반 국내 온라인 강의 추천 서비스입니다.  
자연어로 질문하면 학습 로드맵과 맞춤 강의를 추천해드립니다.

## 🚀 바로 실행하기

**👉 [https://2026-ai-service-project.vercel.app](https://2026-ai-service-project.vercel.app)**

브라우저에서 위 주소에 접속하면 바로 사용할 수 있습니다.  
별도 설치 불필요합니다.

## 💡 사용 방법

1. 위 주소 접속
2. 검색창에 자연어로 질문 입력
   - 예: `토익 처음 시작하는데 돈이 없어요`
   - 예: `파이썬 머신러닝 입문, 5만원 이하`
   - 예: `일러스트 독학하고 싶어요`
3. AI가 학습 로드맵과 단계별 강의를 추천
4. 마음에 드는 강의 선택 후 이메일로 받기

## 🛠 기술 스택

| 구분 | 기술 |
|------|------|
| 프론트엔드 | React 18 + TypeScript + Tailwind CSS |
| 백엔드 | FastAPI + Python |
| AI | OpenAI GPT |
| DB | Supabase |
| 배포 | Vercel (프론트) + Railway (백엔드) |
